# employee-tracker
    application views and manage departments, roles, and eployees for the company for a better 
    and organize way to plan your business. 
    prompt a command-line application that accepts user input
    the useris presented with the following options: view all departments, view all roles, view all employees, add a department, add a role, add an employee, and update an employee role
    if choose to view all departments
    the presented with a formatted table showing department names and department ids
    when the user choose to view all roles
    you are presented with the job title, role id, the department that role belongs to, and the salary for that role
    when the user choose to view all employees
    you are then presented with a formatted table showing employee data, including employee ids, first names, last names, job titles, departments, salaries, and managers that the employees report to
    when the user choose to add a department
    then you are prompted to enter the name of the department and that department is added to the database
    when the user choose to add a role
    then you are prompted to enter the name, salary, and department for the role and that role is added to the database
    when the user choose to add an employee
    then the user is prompted to enter the employeeâ€™s first name, last name, role, and manager, and that employee is added to the database
    when you choose to update an employee role
    then the application is prompted to select an employee to update and their new role and this information is updated in the database
